# Abilator Brand Assets

Offisielt logobibliotek og merkevareressurser for Abilator AS og undermerker.

> **Bruk alltid jsDelivr CDN-URL-er** (ikke raw.githubusercontent.com) for å vise logoer i nettlesere og AI-verktøy.

---

## Slik bruker du logoene

Bytt ut `FILNAVN` med ønsket fil fra tabellene under:

```html
<img src="https://cdn.jsdelivr.net/gh/abilatoras/brand-assets@main/FILNAVN" height="40">
```
